# gateway_pro 启动问题排查记录

## 记录时间

- 2026-02-27 至 2026-02-28

## 已定位并解决的问题

1. MySQL 库不存在
- 现象：`Unable to determine Dialect without JDBC metadata`
- 根因：底层异常 `Unknown database 'jarvis'`
- 处理：创建 `jarvis` 库

2. MySQL 密码未注入
- 现象：仍报 Hibernate 方言无法确定
- 根因：底层异常 `Access denied for user 'root'@'localhost' (using password: NO)`
- 处理：在 IDEA Run Configuration 注入 `MYSQL_PASSWORD`

3. Redis 未监听 6379
- 现象：`RedisConnectionFailureException: Unable to connect to Redis`
- 根因：`Connection refused: /127.0.0.1:6379`
- 处理：`brew services start redis`，`redis-cli ping` 返回 `PONG`

4. 端口占用
- 现象：应用启动失败
- 根因：`Port 8080 was already in use`
- 处理：释放 8080 或改端口启动

## 当前稳定启动前检查

```bash
# MySQL
/usr/local/mysql/bin/mysql --protocol=TCP -hlocalhost -P3306 -uroot -p -e "SHOW DATABASES LIKE 'jarvis';"

# Redis
brew services start redis
redis-cli -h 127.0.0.1 -p 6379 ping
```

## IDEA 关键配置

- Active profiles: `dev`
- Environment variables:
  - `MYSQL_PASSWORD=你的MySQL密码`
  - `SPRING_PROFILES_ACTIVE=dev`（可选，Active profiles 已填时可不填）
